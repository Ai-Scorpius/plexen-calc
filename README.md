# PLEXEN Scientific

A browser-based scientific calculator: exact-math engine (rational + surd
arithmetic, not floats), natural textbook display, and an installable
offline PWA. Vanilla HTML/CSS/JS — no runtime dependencies, no build step.

Serve the directory over HTTP and open it:

```
npx serve .
```

Bundled font: Noto Serif Italic (SIL Open Font License 1.1) — see
`assets/fonts/OFL.txt`.
